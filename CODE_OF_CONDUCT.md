# Code of Conduct

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Standards

Examples of behavior that contributes to a positive environment:

- Being respectful of differing viewpoints and experiences
- Gracefully accepting constructive criticism
- Focusing on what is best for the community

Unacceptable behavior includes harassment, intimidation, abusive comments, and publishing private information without permission.

## Enforcement

Project maintainers may remove, edit, or reject contributions that do not align with this Code of Conduct.
